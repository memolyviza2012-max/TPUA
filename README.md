# 🅰️ TPUA (PUA Font Mapping Tool)

TPUA คือโปรแกรมขั้นเทพสำหรับการ **แก้ปัญหาสระลอย/สระจมในเกม** โดยใช้วิธี Private Use Area (PUA) Mapping ซึ่งเป็นการแมปปิ้งตัวอักษรไทยที่มีปัญหา (เช่น ไม้โท, สระอิ) ไปยังตำแหน่งตัวอักษรว่างในฟอนต์ เพื่อให้เอนจิ้นเกมแสดงผลสระได้ตรงจุดเป๊ะๆ

## 🌟 ฟีเจอร์หลัก (Key Features)
- **Smart PUA Mapping:** แปลงข้อความภาษาไทยธรรมดา ให้กลายเป็นข้อความรหัส PUA อัตโนมัติ
- **Font Shift Adjustment:** ปรับลด/เพิ่มระดับความสูงของสระวรรณยุกต์ได้อย่างละเอียด
- **Batch Processing:** แปลงไฟล์ข้อความทั้งเกม (นับหมื่นบรรทัด) ให้เป็น PUA ภายในไม่กี่วินาที
- **Real-time Preview:** ดูตัวอย่างการซ้อนทับของสระก่อนที่จะนำไปใช้จริงในเกม

## ⚙️ วิธีติดตั้ง (Installation)
1. ติดตั้งผ่าน **THub Launcher** สะดวกที่สุด (ไปที่เมนู Flagship Products)
2. หรือดาวน์โหลดจากหน้า [Releases](https://github.com/memolyviza2012-max/TPUA/releases) ของ Repository นี้

## 🚀 วิธีใช้งานเบื้องต้น (How to Use)
1. นำไฟล์ Font ของคุณไปจัดการใส่ PUA Glyph (แนะนำให้ใช้ร่วมกับ TGlyph)
2. เปิดโปรแกรม TPUA
3. นำไฟล์ข้อความภาษาไทยที่แปลเสร็จแล้ว (จาก TStudio) โหลดเข้าโปรแกรม
4. เลือกรูปแบบ PUA Offset ที่ตรงกับฟอนต์ของคุณ
5. กด **Convert to PUA** โปรแกรมจะเปลี่ยนสระบน-ล่างให้เป็นโค้ด PUA
6. นำไฟล์ที่ผ่านการแปลงแล้วไปแพ็กเข้าเกม สระจะไม่ลอยอีกต่อไป!
