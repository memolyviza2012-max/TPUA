# 🅰️ TPUA (PUA Font Mapping Tool) - คู่มือการใช้งานฉบับสมบูรณ์

TPUA เป็นอาวุธลับของนักแปลเกมไทย เอาไว้ปราบปรามบอสที่ชื่อว่า **"สระลอย / สระจม"** ในเกมที่เอนจิ้นไม่รองรับการจัดเรียงภาษาไทยที่ซับซ้อน (เช่น Unreal, Unity)

## 🌟 ฟีเจอร์ทั้งหมดแบบเจาะลึก
1. **Advanced PUA Engine:** อัลกอริทึมวิเคราะห์สระและวรรณยุกต์ไทย (เช่น ญ ฐ ที่มีสระอุอู, วรรณยุกต์ที่ซ้อนบนสระอิอี) และทำการ Shift ไปยังตำแหน่ง Private Use Area (PUA) อัตโนมัติ
2. **Batch Text Conversion:** ไม่ว่าเกมจะมีบทสนทนาเป็นแสนบรรทัด TPUA สามารถแปลงข้อความทั้งหมดให้รองรับฟอนต์ PUA ได้ในเวลาไม่ถึง 2 วินาที
3. **Vertical Shift Control:** สามารถระบุค่าระยะร่น (Offset) ของฟอนต์แต่ละตัวได้ เพราะฟอนต์แต่ละตระกูลมีสเกลความสูงไม่เท่ากัน
4. **Seamless TStudio Integration:** โหลดไฟล์ `.csv` ที่ Export มาจาก TStudio เข้ามาจัดการได้ทันที

## 🚀 วิธีใช้งานทีละขั้นตอน (Step-by-Step)
1. **เตรียมฟอนต์ PUA:** คุณต้องนำไฟล์ `.ttf` ไปทำ PUA Glyph ผ่านโปรแกรม TGlyph ก่อน
2. **นำเข้าข้อความแปล:** เปิด TPUA แล้วลากไฟล์แปลภาษาของคุณ (ที่แปลเสร็จแล้ว) เข้ามาในโปรแกรม
3. **เลือกรูปแบบ PUA Profile:** เลือก Profile ให้ตรงกับฟอนต์ที่คุณใช้ (เช่น Profile A สำหรับฟอนต์ตระกูล DB, Profile B สำหรับฟอนต์ตระกูล Sarabun)
4. **พรีวิว:** ดูในหน้าต่างพรีวิวว่าตัวอักษร "ป่า", "น้ำ", "หญิง" ซ้อนทับกันถูกต้องหรือไม่
5. **เริ่มแปลง:** กดปุ่ม **Execute PUA Mapping**
6. **นำไปใช้งาน:** โปรแกรมจะเซฟไฟล์ใหม่ที่มีการเข้ารหัส PUA แล้ว (อ่านด้วยตาเปล่าจะเป็นภาษาต่างดาว) ให้นำไฟล์นี้ไปแพ็กเข้าเกม!
